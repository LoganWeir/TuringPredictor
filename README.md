This python classifier applies Logistic Regression to a set of homogenized transcripts from 
the annual Loebner Prize in Artificial Intelligence.

This code uses:
numpy
pandas
scipy.sparse
sklearn
	-LogisticRegression
	-CountVectorizer
	-cross_val_score
